# INSIGHT

각종 테스트를 진행하고 그 결과를 저장하여
성향을 탐구할 수 있는 웹 서비스입니다.

UIUX 디자인부터 프론트, 백엔드까지
웹 개발 전반을 직접 구현하는 것을 목표로 제작했습니다.

현재 테스트는 '간단 MBTI 검사' 한 가지만 존재합니다.

------

## 🖥️ 서비스 주요 기능

- 회원가입 / 로그인
- 메인화면 테스트 목록 조회
- 테스트 진행 및 결과 계산
- 테스트 결과 저장
- 좋아요 기능
- 마이페이지
  - 완료한 테스트 목록
  - 좋아요한 테스트 목록
- 404 에러 페이지 처리

## 📁 프로젝트 구조

test
├─ frontend
│ ├─ assets

│ ├─ styles
│ └─ scripts
│

## 🔑 주요 기능 설명

### 1. 인증 (회원가입 / 로그인)
- JWT 기반 인증 방식
- 로그인 성공 시 토큰을 저장하여 사용자 상태 유지

### 2. 테스트 진행 및 결과 계산
- 테스트 질문과 선택지는 서버에서 조회
- 사용자의 응답 전송 시 서버에서 결과 계산
- 결과는 DB에 저장되어 이후 재조회 가능

### 3. 마이페이지
- 사용자 정보 조회
- 좋아요한 테스트 목록 조회
- 완료한 테스트 결과 목록 조회
- 로그아웃
